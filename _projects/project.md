# Présentation de Klivio

Klivio est l'un de mes premiers projets. C'est une plateforme que j'ai développée pour coller au projet demandé à la web académie, un long projet regrouppant CSS et HTML (et un peu tailwind)

### Détails Techniques
- **Technologies :** HTML, CSS, Tailwind
- **Rôle :** Développeur Fullstack
- **Objectif :** Créer une interface utilisateur fluide et rapide avec prix, avis et commentaires.
